<a name="english"></a>
## :gb: English

### Overview

Manages caregivers and families, intelligently matching profiles with an automatic algorithm. Uses Node.js and SQLite database.

### Features

- **Smart Matching** - Automatic algorithm pairing
- **Profile Management** - Caregiver and family profiles
- **Database Storage** - SQLite persistence
- **Report Generation** - Match reports
- **Search Filters** - Find suitable caregivers

### Quick Start

```bash
git clone https://github.com/fracabu/badanti-apify-report.git
cd badanti-apify-report
npm install
npm start
```
